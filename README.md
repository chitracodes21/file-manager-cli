# 📁 File Manager CLI

A simple and interactive **Node.js Command Line File Manager** that allows you to manage files and folders directly from the terminal.

---

## 🚀 Features

- 📁 Create folders
- 📄 Create files with optional content
- ❌ Delete files and folders
- ✏️ Rename files and folders
- 📂 List directory contents
- 🎯 Interactive CLI menu system
- 🎨 Clean terminal UI using Chalk

---

## 🛠 Tech Stack

- Node.js (fs/promises)
- Readline (CLI input handling)
- Chalk (terminal styling)
- ES Modules (import/export)

---

## ⚙️ Installation

```bash
git clone https://github.com/chitracodes21/file-manager-cli.git
cd file-manager-cli
npm install
```

---

## ▶️ Run the Project

```bash
node app.js
```
